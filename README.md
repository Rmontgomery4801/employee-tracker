# Employee Tracker

# Description

This is a command-line application that can manage a company's employee database, using Node.js, Inquirer, and MySQL.

![screenshot](assets/images/screenshot.png)



# Installation & Usage

```python
npm init
npm i
npm i mysql mysql2
npm i console.table
npm i inquirer inquirer-confirm
```
